## 판정 기준

### 🚫 제재 — 아래 중 하나라도 해당하면 무조건 제재

**[성적 노출 — 청소년보호법]**
- 성기·음부·항문 노출 (실사·삽화·피규어 모두 동일 적용)
- 성행위 묘사 (실사·삽화 모두)
- 스커트·속옷을 들어올려 하의 부위 노출
- 성적 포즈와 노출이 결합된 이미지
- 나체 피규어 (가슴·하체 전체 노출)
- 네글리제·시스루 등 비치는 소재로 실질적 노출 수준인 경우
- 하의 없이 상의만 착용한 피규어
- 가슴 노출이 있는 실사 착용샷 (란제리·코스튬·속옷 포함)
- 음란물·성인DVD·성행위 도구 이미지

**[아동·청소년 — 아청법]**
- 아동·청소년으로 보이는 캐릭터의 성적 연출·노출
  (성인 외형이라도 전체적으로 미성년자 인상이면 동일 적용)

**[잔혹·폭력 — 청소년보호법 시행령 제7조]**
- 신체 절단·훼손을 구체적으로 묘사한 것
- 자해·자살을 미화하거나 구체적으로 묘사한 것
- 피·상처가 광범위하고 자극적으로 연출된 것
  (전신 혈흔, 링거백에 피, 봉합 흔적 전신 등)
- 잔혹한 폭력·살인을 미화한 것

### ✓ 정상 — 아래에 해당하면 정상

- 일반 의류·잡화·전자기기 등 명백히 무해한 상품
- 속옷·수영복이라도 마네킹·일반 착용샷으로 선정성 없음
- 성인용품이지만 포장만 보이고 성적·잔혹 이미지 없음
- 화장품·향수 등 성인 대상이어도 노출 없음
- 원작 캐릭터 의상 그대로이며 성적 연출 없는 피규어
  (예: 나미 비키니·수영복 — 복부·다리 노출이나 성적 연출 없음)
- 교복·일상복 피규어로 노출 없음
- 상의 탈의라도 성적 연출 아닌 경우 (단, 가슴 노출 없어야 함)
- 호러·다크 콘셉트이나 신체훼손 수준 아닌 것
- 정식 발매 트레이딩 카드·굿즈의 비키니·수영복 일러스트
  (성기 노출 없고 과도한 성적 연출 없는 경우)
- 공식 라이선스 수집품으로 카테고리·맥락이 명확한 상품
- 속옷 상품 단독 촬영 (마네킹 또는 상품만, 신체 노출 없음)
- 실사 인물 착용샷이라도 노출 없는 일반 의류·가발·악세서리

### ? 고민 — 진짜 경계선 케이스만 (전체 판정의 20% 이내)

**[성적]**
- 란제리·속옷 착용샷인데 노출 수위 애매
  (가슴 노출 없고 다리·복부 노출만 있는 경우)
- 가슴 노출 수위가 높고 성적 의도가 있으나
  성기 노출은 없는 피규어
- 가터벨트·망사스타킹 등 성적 연출이 강하나
  직접 노출은 없는 피규어
- 비키니·수영복 피규어 중 노출 면적이 크거나
  성적 연출 있는 경우
- 치마를 들어올리는 구도로 속옷이 보이는 피규어
- 성인용품 패키지에 성적 일러스트 있는 경우
- 성인 향 명칭("Bunny Bikini Ver." 등)이나
  포장만 보이는 경우
- 마네킹 착용샷으로 신체 노출 없는 성인 코스튬

**[잔혹]**
- 상처·피가 부분적으로 있으나 자극성 낮은 것
- 공포·호러 콘셉트이나 신체훼손 수준 아닌 것

### 도서·만화류 판단 기준

1순위: 표지 이미지 직접 판단
  - 표지에 성적 노출·잔혹 묘사 있으면 → 제재
  - 표지 무해하면 → 2순위로

2순위: 청소년 유해매체 지정 여부
  - 여가부 청소년 유해매체 공식 지정 → 제재
  - 미지정 → 정상

3순위: 플랫폼 등급 (참고용)
  - 플랫폼마다 버전·등급이 다를 수 있어
    단행본 판단에 직접 적용 어려움
  - 단, 모든 플랫폼이 19세 이상으로
    일치하는 경우 → 제재 참고

### 판단 시 주의사항

- 피규어·삽화라도 실사와 동일 기준 적용
- 아동·청소년 외형 캐릭터는 더 엄격하게 적용 (아청법)
- 실사 인물 착용샷은 상품 카테고리·제목 함께 확인
  (얼굴 블러 처리 자체는 제재 기준 아님)
- 의류·가발·악세서리 착용샷은 노출 없으면 정상
- 포장만 보이고 내용물 이미지 없으면 → 고민
- 일반 상품·의류·잡화 → 무조건 정상
- 명백한 제재 사유 → 고민 없이 즉시 제재
- 이미지가 흐리거나 작아도 보이는 것 기준으로 단호하게 판단
- 상품 페이지 전체 캡처 시 상품명·카테고리·설명까지 함께 판단
